## Project Run

### 1. Clone the repo:

```bash
git clone https://github.com/cydexx/uber-clone.git
```

### 2. Download dependencies:

```bash
npm i
#or
yarn
```

### 3. Run the project:

```bash
npx expo start
```

### 4. See the result :

- IOS: Scan the QR code via camera app and open expo go.
- IOS: Scan the QR code via expo go app.
